Province
------------

Provinces of addresses are represented by the Province model.

+-----------------+-------------------------------------------------------------------------+
| Property        | Description                                                             |
+=================+=========================================================================+
| name            | Province name                                                           |
+-----------------+-------------------------------------------------------------------------+
| code            | Province code (abbreviation)                                            |
+-----------------+-------------------------------------------------------------------------+
| country         | Country of the province                                                 |
+-----------------+-------------------------------------------------------------------------+
