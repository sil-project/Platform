=========================
Domaine du composant User
=========================

-------
Modèles
-------

Role
----

Le modèle Role représente l'entité qui permet de construire la hiérarchie des rôles pouvant être assignés à un utilisateur.

+-----------------+-------------------------------------------------------------------------+
| Propriété       | Description                                                             |
+=================+=========================================================================+
| name            | nom du rôle                                                             |
+-----------------+-------------------------------------------------------------------------+
| parent          | rôle parent                                                             |
+-----------------+-------------------------------------------------------------------------+
| children        | rôles enfants                                                           |
+-----------------+-------------------------------------------------------------------------+
