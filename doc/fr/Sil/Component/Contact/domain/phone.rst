Phone
------------

Les téléphones des Contacts sont représentés par le modèle Phone.

+-----------------+-------------------------------------------------------------------------+
| Property        | Description                                                             |
+=================+=========================================================================+
| number          | Numéro de téléphone                                                     |
+-----------------+-------------------------------------------------------------------------+
| type            | Type du téléphone                                                       |
+-----------------+-------------------------------------------------------------------------+
| contact         | Contact auquel le téléphone appartient                                  |
+-----------------+-------------------------------------------------------------------------+
